Test Deteksi Kepribadian

Jawab pertanyaan berikut untuk mengetahui apakah Anda lebih introvert atau ekstrovert!

1. Saya merasa nyaman saat menghabiskan waktu sendiri.

2. Saya lebih suka berbicara dalam kelompok besar.

3. Saya cenderung berpikir dulu sebelum berbicara.

4. Saya lebih suka menghadiri pesta besar daripada bersantai di rumah.

5. Saya merasa kelelahan setelah banyak berinteraksi sosial.

6. Saya merasa senang menjadi pusat perhatian.

7. Saya lebih suka bekerja sendiri daripada bekerja dalam tim.

8. Saya sering mengambil inisiatif untuk memulai percakapan.

9. Saya lebih suka mendengarkan daripada berbicara dalam diskusi.

10. Saya merasa lebih produktif di tempat yang ramai dan penuh energi.
